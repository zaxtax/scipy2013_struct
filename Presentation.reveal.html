Cavaets

Software for performing Structured Prediction

This talk will use PyCRFsuite and Pystruct

What is PyCRFsuite?

  • A python wrapper of CRFsuite written by Jake Vanderplas

  • CRFsuite is a linear-chain CRF library by Naoaki Okazaki

What is PyStruct

  • A python wrapper around several solvers for models like CRF by Andreas Mueller

  • This includes libdai, AD3, cvxopt, QPBO, OpenGM and others

What is a CRF?

  • Basically a way to model the dependencies between our features and labels

    (('pos', -1), ('pos',  0), ('pos',  1)),
    (('pos',  0), ('pos',  1), ('pos',  2)),
    )


import crfutils

def feature_extractor(X):
    # Apply attribute templates to obtain features (in fact, attributes)
    crfutils.apply_templates(X, templates)
    if X:
        # Append BOS and EOS features manually
        X[0]['F'].append('__BOS__')     # BOS feature
        X[-1]['F'].append('__EOS__')    # EOS feature
